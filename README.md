# Cybersecurity Home Lab

## Overview
This repository documents my hands-on cybersecurity home lab as I build practical skills in networking, Linux, traffic analysis, ethical hacking, and system defense.

## Lab Goals
- Build practical cybersecurity skills outside the classroom
- Practice Linux commands and system administration
- Analyze network traffic using Wireshark
- Explore Kali Linux tools in a legal lab environment
- Document GNS3 networking labs
- Practice basic switch, router, and Raspberry Pi configurations

## Tools & Technologies
- Kali Linux
- Raspberry Pi
- Wireshark
- GNS3
- Linux Terminal
- Cisco/Router/Switch Concepts
- Python
- GitHub

## Lab Sections
### 1. Linux Commands
Basic Linux commands practiced for cybersecurity and networking.

### 2. Wireshark Packet Analysis
Network traffic captures and packet analysis notes.

### 3. Raspberry Pi Setup
Documentation of Raspberry Pi configuration and cybersecurity use cases.

### 4. GNS3 Networking Labs
Virtual networking labs for routing, switching, and topology practice.

### 5. Cybersecurity Notes
Key lessons from ethical hacking, incident response, intrusion detection, and system defense.

## What I Am Learning
Through this lab, I am improving my ability to troubleshoot networks, analyze traffic, understand Linux systems, and apply cybersecurity concepts in a practical environment.
